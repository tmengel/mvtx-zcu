- bitvis_vip_i2c      : An example BFM that uses UVVM
  - doc               : Quick Reference for BFM and VVC
  - script            : Compile scripts
  - src               : I2C BFM and VVC source code
IMPORTANT
This is a simplified Verification IP (VIP) for I2C.
The given VIP complies with the basic I2C protocol and thus allows a 
normal access towards an I2C interface. This VIP is not an I2C protocol checker.
For a more advanced VIP please contact Bitvis AS at support@bitvis.no
